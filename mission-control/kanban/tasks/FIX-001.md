## Fix Applied

**Root cause**: Langfuse v3 ignores generic `S3_REGION`/`S3_ENDPOINT` environment variables.
It requires a full set of per-scope prefixed vars:

```yaml
LANGFUSE_S3_EVENT_UPLOAD_REGION:            us-east-1
LANGFUSE_S3_EVENT_UPLOAD_ENDPOINT:          http://minio:9000
LANGFUSE_S3_EVENT_UPLOAD_ACCESS_KEY_ID:     minio
LANGFUSE_S3_EVENT_UPLOAD_SECRET_ACCESS_KEY: <secret>
LANGFUSE_S3_EVENT_UPLOAD_FORCE_PATH_STYLE:  "true"
# same pattern for LANGFUSE_S3_MEDIA_UPLOAD_*
```

Both `langfuse-web` and `langfuse-worker` updated in `mission-control/docker-compose.yml`.
